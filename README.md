# pizzaria_web
O projeto consiste em um site moderno e responsivo para uma pizzaria, com o objetivo de divulgar o cardápio, promoções e facilitar pedidos online. O site apresenta uma interface intuitiva, com seções de sabores, bebidas, combos, avaliações de clientes e informações de contato. Também conta com integração a WhatsApp para pedidos rápidos e um painel simples para atualização de produtos e preços.
